Desenvolvimento de API REST: Construir, Testar e Solucionar Problemas

Aprenda a construir, configurar, testar e depurar APIs REST usando exemplos práticos

As APIs REST são a base das aplicações e serviços web modernos. Neste curso, você aprenderá a projetar, desenvolver, testar e solucionar problemas de APIs REST passo a passo usando ferramentas práticas e exemplos do mundo real.

Começando pelos fundamentos de HTTP e JSON, você construirá uma API REST totalmente funcional, entenderá como diferentes métodos HTTP funcionam e aprenderá a testar sua API usando o Postman.

O que você vai aprender

  • Os fundamentos do HTTP e da arquitetura de APIs REST
  • Como APIs REST se comunicam usando JSON
  • Como configurar e criar um projeto de API REST
  • Construção de modelos e controladores para endpoints de API
  • Entender os métodos GET, POST, PUT, DELETE e PATCH
  • Criar rotas personalizadas e filtrar resultados da API
  • Evitar registros duplicados em operações POST
  • Testar APIs usando o Postman
  • Automatizar testes de API e usar variáveis no Postman
  • Solucionar problemas comuns em APIs REST
